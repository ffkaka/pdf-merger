# AGENTS.md

## 목적
이 저장소의 에이전트는 여러 PDF 파일이 있는 폴더에서 여러 키워드(예: `AAA-BBB1`, `AAA-BBB2` ... `FFF-FFF9`)를 받아, 키워드를 포함한 PDF를 찾고 하나의 PDF로 병합한다. 병합 결과 파일명은 키워드와 동일하게 저장한다.

## 필수 스킬
- pdf (read/create/review PDF with layout fidelity, rendering checks 포함)

## 스킬 자동 설치
아래 스킬이 설치되어 있지 않다면, 다음 명령으로 설치한다.
- codex skill install pdf

## 기본 작업 규칙
- 새 PDF 생성은 `output/pdf/` 아래에 저장한다.
- 중간 산출물은 `tmp/pdfs/` 아래에 둔다.
- 레이아웃 확인이 필요한 경우 페이지 렌더링 후 육안 검수한다.
- 키워드 검색은 PDF 텍스트를 추출해 수행한다.
- 키워드가 여러 개일 경우, 키워드별로 각각 병합 결과를 생성한다.
- 병합 결과 파일명은 키워드와 동일하게 한다. 예: `output/pdf/AAA-BBB1.pdf`

## 표준 워크플로우
1. 입력 폴더 스캔: PDF 목록을 수집한다.
2. 키워드 검색: 각 PDF에서 텍스트를 추출해 키워드 포함 여부를 판정한다.
3. 병합 대상 선정: 키워드를 포함한 PDF만 목록화한다.
4. 병합: 키워드별로 대상 PDF를 지정된 순서로 하나의 PDF로 합친다.
5. 저장: 결과 파일명은 키워드와 동일하게 한다.
6. 검수: 병합된 PDF를 열어 누락/깨짐 여부를 확인한다.

## 산출물 규칙
- 최종 결과물 파일명은 작업 목적을 반영한다. 예: `환경설정.pdf`, `merged_user_guide.pdf`
- 생성/병합 결과는 항상 `output/pdf/`에 둔다.
